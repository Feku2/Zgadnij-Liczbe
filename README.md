
# "Zgadnij liczbę" w języku C++

### Wymagania:

*-ekran powitalny skąd możemy:
*  a) rozpocząć nową grę
*  b) sprawdzić aktualne TOP5 (tylko gdy już jakieś są!)
*-na ekranie wyników wyświetlamy TOP5 najlepszych, gdzie będziemy widzieć nazwę użytkownika, ilość prób, oraz poziom trudności. Z ekranu wyników możemy wrócić do ekranu powitalnego.
*-przed rozpoczęciem gry należy wybrać jeden z poziomów trudności np. łatwy (1-50), średni (1-100), trudny (1-250)
*-podczas gry na ekranie zawsze wyświetla się aktualny numer próby
*-w przypadku braku podania prawidłowej liczby, na ekranie dostaniemy komunikat czy liczba jest za mała czy za duża i zostanie uruchomiona kolejna próba
*-w przypadku trafienia prawidłowej liczby dostaniemy komunikat o podanie swojego imienia.
*-Po wpisaniu imienia zostaniemy przekierowani do ekranu powitalnego

### Dodatkowo:
*-losowe komunikaty zamiast jednego stałego, gdy nie trafimy liczby
*-ekran wyników podzielony na poziomy trudności, które możemy przełączać
*-tryb zakładu, o który zapyta gra po wybraniu poziomu trudności - jeśli tak, to gracz podaje maksymalną liczbę prób w następnej rozgrywce
*-stwórz prymitywny "interfejs graficzny"
*-zapis i wczytywanie wyników do pliku (nice to have)
*-użycie gita do rozliczenia pracy
